# Command line tools written by swift

**proof-of-concept**

This cli written by [Swift](https://developer.apple.com/swift/ "Swift").

## Usage

```sh
git clone
./git-current-branch.swift

# or
xcrun swift -i -sdk $(xcrun --show-sdk-path --sdk macosx) git-current-branch.swift
```
